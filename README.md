# Clash 代理订阅

自动更新的 Clash 代理订阅服务。

## 订阅地址

```
https://raw.githubusercontent.com/rico-cell/clash-proxy-sub/main/clash_sub.yaml
```

## 功能

- ✅ 每 30 分钟自动更新
- ✅ 293+ 节点，覆盖 75+ 国家
- ✅ 按国家分组，带国旗 emoji
- ✅ 自动测速选择最快节点

## 使用方法

1. 复制上面的订阅地址
2. 在 Clash Verge / Clash for Windows 中添加订阅
3. 启用订阅即可使用

## 更新日志

配置文件由 GitHub Actions 自动更新。
